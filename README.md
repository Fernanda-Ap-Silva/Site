# 🌐 Projeto HTML Semântico – Bootcamp Santander 2025

Este projeto foi desenvolvido como parte do módulo **"Entendendo HTML Semântico"** do Bootcamp Santander 2025. O objetivo é criar uma página HTML utilizando boas práticas de estruturação de conteúdo e acessibilidade, com foco em semântica, organização e usabilidade.

## 📚 Conteúdo aprendido e aplicado

Durante as aulas práticas e teóricas, aplicamos:

### ✅ Tags de Texto e Formatação:
- `<h1>` a `<h6>` – Hierarquia de títulos
- `<p>`, `<mark>`, `<small>`, `<i>`, `<u>`, `<strong>`
- `<sub>`, `<sup>`, `<del>`, `<abbr>`

### ✅ Listas e Estrutura:
- `<ol>`, `<ul>`, `<li>`
- `<hr>`, `<blockquote>`, `<q>`

### ✅ Estrutura Semântica do HTML5:
- `<main>`, `<header>`, `<footer>`
- `<section>`, `<article>`, `<aside>`, `<nav>`

### ✅ Mídias e Recursos:
- `<img>`, `<audio>`, `<video>`, `<track>`, `<picture>`, `<figure>`, `<figcaption>`
- Atributos `alt`, `controls`, `default`, `cite`

## 🧭 Objetivo do Projeto

Criar uma página HTML aplicando:
- Tags semânticas corretamente
- Acessibilidade com uso de atributos e estrutura lógica
- Organização com separação clara de conteúdos

## 🧩 Recursos extras

- Índice de aulas com botão de voltar
- Links úteis e vídeos de apoio sobre HTML e semântica
- Explicações didáticas como se estivesse ensinando para alguém iniciando

## 📌 Como visualizar

Você pode abrir o arquivo `index.html` no navegador ou usar uma IDE como o VS Code com extensão Live Server.
