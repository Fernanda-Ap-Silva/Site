Aula 1: Introdução ao HTML Semântico

HTML semântico é uma forma de escrever código onde usamos tags que descrevem o significado do conteúdo que está dentro delas.

Por exemplo, usamos a tag <header> para o cabeçalho da página, <nav> para a área de navegação e <main> para o conteúdo principal.

"O HTML semântico ajuda a melhorar a acessibilidade e SEO dos sites." – MDN Web Docs

Aula 2: Tags Importantes e Estruturais

Vamos conhecer algumas tags que você deve usar para estruturar seu site:

  • <header> — Cabeçalho
  • <footer> — Rodapé
  • <article> — Artigo ou conteúdo independente
  • <section> — Seções de conteúdo
  • <aside> — Conteúdo lateral, como anúncios ou notas

Exemplo: O uso correto das tags ajuda pessoas que usam leitores de tela e também melhora o SEO.
